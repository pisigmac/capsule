# Email

## Configuration
| Variable | Description |
|----------|-------------|
| `SMTP_HOST` | SMTP server |
| `SMTP_PORT` | SMTP port |
| `SMTP_USER` | Username |
| `SMTP_PASS` | Password |
| `FROM_EMAIL` | Sender address |

## Templates
- Welcome email
- Weekly digest
- Stale capsule reminder
- Password reset (future)

## Providers
- SendGrid
- AWS SES
- Mailgun

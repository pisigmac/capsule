# Pricing

## Philosophy

Capsule is free for individual use. Team features require a subscription.

## Tiers

### Free — $0/month

**For:** Individual developers, open-source contributors

**Includes:**
- Unlimited local capsules
- Full-text search
- Context composition
- File system sync
- CLI access
- SQLite database
- Open source (self-host)

**Limits:**
- Single user
- Local storage only
- No web UI
- No team sharing
- No sync across devices
- Community support only

### Pro — $9/month

**For:** Professional developers, consultants

**Includes everything in Free, plus:**
- Cloud sync across devices
- Web UI for browsing
- Capsule templates
- Import from Notion, Obsidian
- Export to PDF, HTML
- Git integration
- Priority email support

**Limits:**
- Single user
- 10,000 capsules
- 5GB storage

### Team — $29/month

**For:** Small teams, startups

**Includes everything in Pro, plus:**
- Up to 10 team members
- Shared team workspace
- Real-time collaboration
- Comment threads
- Capsule ownership
- Team analytics
- Slack integration
- Standard support

**Limits:**
- 10 users
- 100,000 capsules
- 50GB storage
- 5 teams

### Enterprise — Custom

**For:** Large organizations, regulated industries

**Includes everything in Team, plus:**
- Unlimited users
- Unlimited capsules
- Unlimited storage
- SSO (SAML, OIDC)
- Audit logs
- Compliance reporting (SOC2, GDPR)
- On-premise deployment
- Custom integrations
- Dedicated support
- SLA guarantee

## FAQ

**Q: Is the open-source version limited?**
A: No. The CLI and local SQLite version is fully functional and unlimited.

**Q: Can I self-host the Team features?**
A: Enterprise includes on-premise deployment. Team features require cloud subscription.

**Q: What happens if I cancel?**
A: Your capsules remain accessible in read-only mode. You can export anytime.

**Q: Do you offer discounts?**
A: Open-source projects, students, and non-profits get 50% off. Contact us.
